# Ribo Downloader

<h4>Features</h4>
<ul>
	<li>Download chunk
	<li>Manager download
	<li>Download Queue
    <li>Multithreaded optimisations
    <li>Pause and resume the downloads
    <li>Threaded, state controlled architecture
    <li>Highly independant modules
    <li>Standalone portable downloader
</ul>
